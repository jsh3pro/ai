위 내용에 대하여 다음 공식으로
유튜브 제목을 10개 추천해줘.
시청자가 제목을 클릭하는 것이 최우선 목표야.

[제목 생성 공식]

A + B + C 구조를 기본으로 할 것.

- A = 감정/훅/충격 발언
- B = 사건/위기/상황
- C = 이유/반전/진실/궁금증

핵심 원칙:
- 클릭 → 이해 → 궁금증 순서로 구성
- 정치·시사 유튜브 스타일 우선
- 실제 유튜브 CTR이 높을 법한 스타일로 생성
- 너무 문어체스럽지 않게
- 실제 사람이 클릭할 만한 제목 우선
- 중장년층 시청자 반응 고려
- 제목마다 감정 흐름이 다르게 생성
- 같은 패턴 반복 최소화

길이 기준:
- A : 5~10자 권장
- B : 10~15자 권장
- C : 10~15자 권장
- 전체 길이 : 32~48자 권장

- 감정
- 충돌
- 반전
- 이유
- 위기
- 숫자
- 충격 발언
- 몰락
- 배신
- 숨겨진 진실
- 내부 폭로
- 여론 반전
- 공포
- 분노
- 통쾌함
- 붕괴
- 갈등
- 초비상
- 결정적 순간

[추천 제목 유형]

1. “대사” + 위기 + 이유형
예:
“A”… B한 진짜 이유

2. “대사” + 충격결과형
예:
“A”… 결국 B됐다

3. 감정 + 사건 + 궁금증형
예:
“A”… B가 터진 숨겨진 이유

4. 숫자 + 위기 + 반전형
예:
숫자 + B… 사람들이 놀란 진짜 이유

5. 위기 + 숨겨진 진실형
예:
B 위기… 내부에서 터진 충격 진실

6. 몰락 + 이유형
예:
B는 왜 공공의 적이 되었나

7. 반전형
예:
“A”… 그런데 상황이 완전히 바뀌었다

8. 배신 + 충격형
예:
“A”… 결국 내부마저 등을 돌렸다

[중요]

- 같은 단어 반복 최소화
- 같은 문장 구조 반복 최소화
- 제목마다 감정 포인트를 다르게
- 제목마다 CTR이 높아질 만한 요소를 반드시 포함
- 썸네일 문구로 확장 가능하게 생성
- 댓글 논쟁 유도가 가능하도록 생성
- 사람들이 “왜?”를 느끼게 만들 것
